À propos¶

Cet atelier de formation de niveau intermédiaire est la suite de l’atelier Premiers pas sur les serveurs de calcul. L’objectif principal est d’explorer un peu plus en profondeur certains aspects clés du calcul informatique de pointe.

Note : les notebooks IPython du présent atelier et les données associées sont publiés sur GitHub.

Table des matières¶

  1. Bien choisir les ressources

    1. Rappel - le calcul haute performance

    2. Analyse des calculs à petite échelle

    3. Extrapoler les ressources nécessaires

    4. Principales différences entre les grappes de calcul

  2. Analyse des tâches

    1. Tâches en attente

    2. Tâches actives

    3. Tâches terminées

  3. Lots de tâches - parallélisme de données

    1. GNU Parallel

    2. Vecteurs de tâches

    3. Le Greedy Launcher Of Small Tasks, ou GLOST

  4. Les espaces de stockage
